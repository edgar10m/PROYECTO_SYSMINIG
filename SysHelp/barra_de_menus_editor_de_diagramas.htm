Barra de menús Editor de diagramas
 
 
Esta barra de menús nos permite acceder a los comandos de Sys-Mining
©
. Algunos de los comandos muestran una pequeña flecha, eso quiere decir que contienen un submenú que se abrirá si mantenemos el cursor del ratón sobre el elemento del menú.
 
Esta barra está disponible únicamente en:
 
  • • 
    • 
    Editor de diagramas
     
Posibilitan el acceder a cada una de las órdenes de Sys-Mining
©
de una forma más rápida.
En el menú puede realizar las siguientes tareas con un Diagrama:
Crear nuevo
 *Dar clic en este menú, opción New diagram,
 *O utilizar el método abreviado del teclado Ctrl+N.
 
Abrir
 *Dar clic en este menú, opción Open diagram,
 *O utilizar el método abreviado del teclado Ctrl+F11.
 
Guardar
 *Dar clic en este menú, opción Save diagram,
 *O utilizar el método abreviado del teclado Ctrl+S.
 *O dar clic en la opción Save as, si es que desea guardarlo con otro nombre o en otra ubucación.
 
Configurar página
 *Dar clic en este menú, opción Page setup y aparece un panel donde puede modificar el tamaño de la hoja, ancho, altura, orientación y margenes.
 
 
Generar Vista previa de impresión
 *Dar clic en este menú, opción Print preview,
 
Imprimir
 *Dar clic en este menú, opción Print,
 
Arriba
Permite copiar, pegar, quitar cambios y poner una imagen de fondo al diagrama.
En el menú puede:
Deshacer: Para cuando quiera quitar un cambio realizado.
 *Dar clic en este menú, opción Undo.
 *O utilizar el método abreviado del teclado Ctrl+Z.
 
Rehacer: Para cuando quiera volver a poner el cambio realizado.
 *Dar clic en este menú, opción Redo.
 *O utilizar el método abreviado del teclado Shift+Ctrl+Z.
 
Cortar elementos seleccionados
 *Dar clic en este menú, opción Cut,
 *O utilizar el método abreviado Ctrl+X
 
Copiar elementos seleccionados
 *Dar clic en este menú, opción Copy
 *O utilizar el método abreviado Ctrl+C
 
Pegar elementos seleccionados
 *Dar clic en este menú, opción Paste
 *O utilizar el método abreviado Ctrl+V
 
Borrar elementos seleccionados
 *Dar clic en este menú, opción Delate
 *O utilizar el método abreviado Ctrl+Dell
 
Seleccionar todo
 *Dar clic en este menú, opción Select all.
 *O utilizar el método abreviado del teclado Ctrl+A.
 
Poner una imágen de fondo
 *Dar clic en este menú, opción Background, opción Load picture, para seleccionar del directorio la que desse..
 *Y para quitar dicha imagen, basta con dar clic en la opción Clear
 
Copiar como imagen
  *Dar clic en este menú, opción Copy as image.
 
Arriba
Permite quitar o añadir complementos al área de trabajo, como reglas o cuadrícula o ajustar el zoom.
En el menú puede:
Ocultar o visualizar los siguientes elementos:
- Regla Izquierda
- Regla superior
- Cuadrícula
 *Dar clic en este menú, y seleccionar la opción que desee.
 
Ajustar el zoom
 *Dar clic en este menú, y seleccionar el porcentaje de zoom que desee.
 
Colocar nodos automáticos
 *Dar clic en este menú, opción Automatic nodes.
 
Visualizar el diagrama por partes
 *Dar clic en este menú, opción Navigator, y se abrirá un panel de lado izquierdo en el cual puede dar clic en el área del diagrama que desea visualizar..
 
Ocultar o mostrar el panel de las capas
 *Dar clic en este menú, opción Diagram layers
 
Arriba
Permite editar la apariencia del diagrama.
En el menú puede realizar las siguientes tareas con un Diagrama:
Poner color en la forma
 *Dar clic en este menú, opción Shape color,
 
Poner color en la línea
 *Dar clic en este menú, opción Line color,
 
Configurar gradiente
 *Seleccioar el o los objetos y dar clic en este menú, opción Gradient setting, y en la ventana que aparece, puede seleccionar dos colores así como la orientación del gradiente.
 
 
Picture
 *Seleccionar el o los objetos, dar clic en este menú, opción Picture y elegir la imagen que desee.
 
 
Cambiar la apariencia de la flecha
 *Seleccioar la o las flechas y dar clic en este menú, opción Source arrow, y seleccionar la que desee.
 *O la opción Target arrow para cambiar el objetivo de la flecha.
 
Cambiar la fuente
 *Seleccionar el o los objetos, dar clic en este menú, opción Font y seleccionar la fuente que desee,
 *O utilizar el método abreviado Ctrl+F.
 
Editar el texto
*Seleccionar el o los objetos, dar clic en este menú, opción Edit text y modificar lo que dice en el objeto que seleccionó,
 *O utilizar el método abreviado F2.
